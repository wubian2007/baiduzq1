# DS足球 - 专业足球赛事分析

## 项目简介

DS足球是一个专业的足球赛事分析应用，基于标准V站的设计和功能进行定制开发。

## 主要特性

- **3D轮播图效果**：使用Swiper.js实现的炫酷3D Coverflow效果
- **专业足球分析**：专注于足球赛事的深度分析和预测
- **响应式设计**：完美适配移动端和桌面端
- **现代化UI**：简洁美观的用户界面设计

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Swiper.js (轮播图库)

## 主要功能

1. **应用展示**：展示DS足球应用的基本信息
2. **3D轮播图**：展示应用截图，支持触摸滑动
3. **下载功能**：提供应用下载入口
4. **弹窗交互**：下载确认和二维码展示
