# DS足球下载站点复刻版

这是对DS足球下载站点的完整复刻，包含了原网站的所有功能和视觉效果。

## 功能特性

### 🎨 视觉设计
- **响应式设计** - 完美适配移动端和桌面端
- **现代化UI** - 采用卡片式设计和圆角元素
- **流畅动画** - 包含页面加载动画和交互动效
- **3D轮播图** - 应用截图的3D轮播展示效果

### 📱 交互功能
- **轮播图控制** - 支持触摸滑动、鼠标拖拽和自动播放
- **智能下载** - 根据设备类型显示不同的下载方式
- **弹窗系统** - 优雅的模态框和二维码弹窗
- **下载反馈** - 完整的下载流程和成功提示

### 🔧 技术实现
- **纯原生技术** - 使用HTML5、CSS3、JavaScript ES6+
- **模块化代码** - 清晰的类结构和事件管理
- **性能优化** - 图片懒加载和动画优化
- **兼容性** - 支持现代浏览器和移动设备

## 功能说明

### 轮播图功能
- **自动播放** - 每3秒自动切换
- **手动控制** - 点击圆点或滑动切换
- **3D效果** - 当前图片居中，其他图片有透视效果

## 自定义配置

### 修改应用信息
在 `index.html` 中修改以下内容：
```html
<h1 class="app-name">DS足球</h1>
<span class="app-version">V1.0.0</span>
<span class="developer">开发者：DS足球团队</span>
<span class="update-date">2025/01/01更新</span>
```

### 更换图片资源
1. 替换 `images/` 目录中的图片文件
2. 保持文件名不变，或修改HTML中的图片路径
3. 建议图片尺寸：
   - 应用图标：100x100px
   - 应用截图：200x350px
   - 横幅图片：按比例缩放

### 调整样式
在 `style.css` 中修改颜色主题：
```css
/* 主色调 */
--primary-color: #e6322e;
--primary-hover: #c40704;
--text-color: #333;
--bg-color: #f5f5f5;
```
